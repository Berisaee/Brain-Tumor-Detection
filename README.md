# Brain Tumor Detection using CNN and PyTorch

## Overview
The Project aims to develop a Convolutional Neural Network (CNN) using the PyTorch framework that can accurately detect and classify Brain Tumors from MRI scans. The CNN will be trained on a large dataset of labeled brain tumor images to learn the patterns and features associated with different tumor types.
The goal of this project is to build a deep learning model capable of classifying brain MRI images into tumor and non-tumor categories with high accuracy.

---

## Features
* Brain MRI image classification using CNN
* Data preprocessing and augmentation
* Dataset splitting into train and validation sets
* PyTorch-based deep learning pipeline
* Model training and evaluation
* Accuracy and loss visualization
* GPU support with PyTorch

## Dataset
The dataset contains brain MRI images categorized into tumor and non-tumor classes.

The dataset is split into:
* 80% Training data
* 20% Validation data

Download here:
https://www.kaggle.com/datasets/preetviradiya/brian-tumor-dataset
